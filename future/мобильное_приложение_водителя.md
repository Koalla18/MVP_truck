# Мобильное приложение для водителя

## Обзор
React Native приложение для Android и iOS для водителей грузовых транспортных средств.

## Технологический стек
- **Framework**: React Native 0.73
- **State Management**: Redux Toolkit
- **Navigation**: React Navigation 6
- **Maps**: react-native-maps (Google Maps / Yandex Maps)
- **Push Notifications**: Firebase Cloud Messaging
- **Offline Storage**: WatermelonDB
- **Background Location**: react-native-background-geolocation

## Основные функции

### 1. Авторизация и профиль
- Вход по номеру телефона + SMS-код
- Биометрическая аутентификация (Face ID / Touch ID)
- Профиль водителя (фото, документы, рейтинг)

### 2. Текущий рейс
- Карта с маршрутом и текущей позицией
- Информация о грузе (вес, габариты, тип)
- Точки погрузки и разгрузки
- Контакты диспетчера и клиента
- Чат с диспетчером

### 3. Контроль времени работы (ФЗ-196)
- Счетчик времени в смене
- Уведомления о необходимости отдыха
- Автоматический учет отработанных часов
- Журнал смен

### 4. Документы
- Электронные ТТН (подпись на экране)
- Фото грузовых документов
- История поездок
- Выгрузка документов в PDF

### 5. Осмотр транспорта
- Чек-лист предрейсового осмотра
- Фото повреждений
- Отметка техсостояния

### 6. Уведомления
- Новый рейс
- Приближение к точке
- Сообщения от диспетчера
- Напоминания об отдыхе
